A collection of complete, working queries for common analytical questions. Copy
one, swap out the figures/degrees that matter for your own question, and run it
across the corpus.



## Cadential soprano descent, quarter notes only, landing on a fermata

A stepwise scale-degree descent restricted to the cantus firmus into the first
scale degree, where the cantizans may be evaded to the 3rd scale degree,
restricted to quarter notes and requiring a fermata on the final note:

```json
{
    "feature": "deg",
    "voices": "soprano",
    "pattern": [
        { "deg": "3", "duration": "4" },
        { "deg": "2", "duration": "4" },
        { "deg": ["1", "3"], "duration": "*", "fermata": true }
    ]
}
```
